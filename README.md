## 1. 댓글 감정 모델 개발
- 유튜브 댓글 데이터를 수집하여 감정을 분류하는 모델을 개발한 프로젝트입니다.
단순히 긍/부정 분류를 넘어, 다양한 감정(기쁨, 기대, 애정, 슬픔, 분노, 공포)을 분류하여 더 정교한 감정 분석을 목표로 하였습니다

## 2. 프로젝트 개요

프로젝트 배경/문제 정의
